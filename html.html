Input/Output Devices

What are I/O Devices?

Input/Output devices are hardware components that allow data to enter and exit a computer system.

Input Devices

  • Keyboard: Used to type text and commands
  • Mouse: Controls cursor position on screen
  • Scanner: Converts physical documents into digital form
  • Microphone: Captures audio input

Output Devices

  • Monitor: Displays visual information
  • Printer: Produces hard copies of digital documents
  • Speaker: Produces audio output
  • Projector: Projects images on a larger screen

Example:

When you type on a keyboard (input), the text appears on the monitor (output). This is a complete I/O operation.

Memory

Types of Computer Memory

Memory is used to store data and programs. There are two main types:

RAM (Random Access Memory)

  • Temporary storage for running programs
  • Loses all data when computer is turned off
  • Much faster than permanent storage
  • Measured in GB (Gigabytes)

ROM (Read Only Memory)

  • Permanent storage for essential system instructions
  • Cannot be modified by users
  • Retains data even when power is off

Hard Drive/SSD

  • Permanent storage for files and programs
  • Much larger capacity than RAM
  • Slower access than RAM but retains data

Memory Comparison:

If a computer has 8GB RAM and a 256GB SSD: RAM holds temporary data while running programs, and the SSD stores all your files permanently.

CPU (Central Processing Unit)

The Brain of the Computer

The CPU executes all instructions and controls all other components of the computer.

Main Functions:

  • Fetch: Retrieve instructions from memory
  • Decode: Understand what the instruction means
  • Execute: Perform the required operation
  • Store: Save the result back to memory

CPU Specifications:

  • Clock Speed: Measured in GHz (Gigahertz) - determines how fast it processes
  • Cores: Multiple cores allow parallel processing
  • Cache: Very fast memory built into the CPU

Example:

A modern CPU with 8 cores running at 3.5 GHz can process 8 instructions simultaneously at 3.5 billion operations per second.

Operating System

What is an Operating System?

An operating system (OS) is software that manages hardware resources and enables user-application interaction.

Main Functions:

  • Resource Management: Allocates CPU, memory, and disk space
  • User Interface: Provides graphical or command-line interface
  • File Management: Organizes and manages files and folders
  • Security: Protects system from unauthorized access
  • Device Control: Manages printers, monitors, and other devices

Common Operating Systems:

  • Windows: Most popular for personal computers
  • macOS: Used in Apple computers
  • Linux: Open-source OS used in servers
  • Android/iOS: Mobile operating systems

Example:

When you click a file, the OS locates it on the hard drive, loads it into RAM, and tells the CPU to open it.

Arrays

                        
Arrays are used to store multiple values of the same data type.

                        
                        

                            
Java Code:

int sum = 0;
for (int i = 0; i < numbers.length; i++) {
    sum += numbers[i];
}
System.out.println("Sum: " + sum);
System.out.println("Average: " + (sum / numbers.length));
